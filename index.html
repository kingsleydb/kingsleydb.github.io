My team conducted research using a functional near infrared spectroscopy (fNIRS) machine; the fNIRS machine used lasers to collect data about a human brain. We used this data as input to a machine learning algorithm that made predictions about the state of the user. Before this research, I never considered that people could interact with computers without explicitly doing anything.
