Brain-Computer Interaction Research Assistant

Tufts University's Human-Computer Interaction Lab

This was a wonderful experience for me because I was able to explore two interests of mine: exploratory computer science research and mindfulness. During my time at the lab, I recieved hands-on experience with cutting-edge Human-Computer Interaction technology, and my research partner was a practicing Zen Buddhist monk prior to starting his Ph.D. in computer science. My technical contribution to the project was developing an adaptable data filter that used regular expressions and an existing Java code base that I translated into Python 3. This filter cleaned all data from the functional near infared spectroscopy (fNIRS) for input to machine learning software. The project team was ultimately able to distingush periods of rest, singular tasks, and multiple tasks from implicit brain data. This study is on-going.
